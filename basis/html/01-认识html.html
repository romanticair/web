Sublime插件及快捷方式的使用:

  • Ctrl + '~' 调出控制台
  • Ctrl + Shift + p 调出安装插件窗口
  • Ctrl + shift + d 快速复制当前行
  • Ctrl + x 快速剪切当前行
  • Ctrl + '+'/'-' 字体大小控制
  • Ctrl + '/' 注释/注释代码
  • Ctrl + shift + ↑↓ 快速转移当前代码行
  • Ctrl + click 多个光标
  • 输html:xt/xs + tab 内部html基本结构就自动填充出来了

    • Ps: 创建文件后改好后缀

    Tips: 插件下载，页面配置，自行慢慢尝试

    知识点:

  • utf-8 字符集, 一个汉字三个字节
  • gb2312/GBK 字符集，一个汉字是两个字节
  • Html: 负责语义、结构
  • Css: 负责样式
  • Javascript: 负责交互

    • HTML 基本语法特性:

  • 1. 在 meta 标签里设置描述、关键字等，有助于优化搜索权重。
  • 2. 对换行和tab和空格不敏感, 且占内存
  • 3. 换行、tab、连续空格都只会折叠成一个空格(空白折叠现象)
  • 4. 标签严格封闭(好习惯)

    • 给 HTML 标签分类

  • 容器级标签 代表：h系列
  • 文本级标签 代表：p系列
  • 图片 img 标签(不需成对出现), alt(alternative-可选), title表示补充说明
  • 超链接 a(anchor-锚)标签, hyper reference(href超链接引用缩写), target-靶(_self自己打开-默认的, _blank则在新窗口打开), title-补充说明

    • Ps: 相对路径/绝对路径都可以
